# [WebPong.net](http://webpong.net/)
<!--# Web Pong-->

An online 1v1 Pong Site!

Game server written in Node.js. UI written in JavaScript, HTML, and CSS.

Hosted on Amazon EC2.

Use the Up and Down arrow keys to move your paddle.
